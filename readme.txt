マルチプレイモードについて：
ゲーム内でオンライン/オフラインを切り替えることが出来ます。
初めからオフラインで起動したい場合は、「offline用」の中身を既存のファイルに上書きしてください。
オンラインモードにすると、外部通信が行われます。
